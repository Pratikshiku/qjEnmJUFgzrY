### 三、功能描述
系统分为（身份）：毕业生、企业、管理员。

**毕业生功能：**
1. 登录
2. 注册
3. 个人中心
4. 个人简历
5. 企业空中宣讲会（收藏、浏览、评论）
6. 岗位应聘、岗位咨询、收藏岗位
7. 添加求职信息
8. 论坛信息（发帖）
9. 线上考试
10. 招聘资讯
11. 我的收藏
12. 我的发布
13. 我的考试记录
14. 错题本
15. 面试回复
